# AGENT OPERATIONS

## Scope Control
- Allowed modules only:
  - Fundamentals Lab
  - Exam Engine
  - Career Launchpad
  - Project Studio (optional minimal stub)
- Do not add unlisted product lines.

## File Change Discipline
- Edit only files required by the spec.
- Do not create duplicate alternatives of same purpose.
- Avoid unnecessary rewrites when content is unchanged.

## Data Rules
- Use local JSON files under `/data` as NoSQL-like database.
- No SQL and no external DB service.
- Load JSON into memory, modify, and write back.
- Keep each JSON file under ~5 MB for MVP.

## Efficiency Rules
- Keep initial implementation minimal but functional.
- Prefer small, focused components.
- Avoid cosmetic rewrites.

## Education Ethics
- Prompt learner attempt before revealing solutions.
- Keep solutions explanatory and stepwise.
- Avoid instant-answer-only experiences.
